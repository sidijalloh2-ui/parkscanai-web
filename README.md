# parkscanai-web

Public marketing site for **ParkScan AI: Sign Reader** — served at [parkscanai.com](https://parkscanai.com) via GitHub Pages.

- `index.html` — the entire site, self-contained (all images/CSS/JS inlined). Edit this file, commit, push → GitHub Pages redeploys automatically.
- `CNAME` — custom domain config for Pages.

The iOS app's source lives in a separate private repo.
